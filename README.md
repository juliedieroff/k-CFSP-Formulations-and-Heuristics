# Implementation of the models and heuristics for the kCFSP

These files contain the explicit and reduced models for the kCFSP as well as the heuristics used to improve solution performance. The "Explicit and Reduced 2 Formulation" file contains the code required to compare the explicit formulation with cut-set constraints to the stronger reduced formulation. The "Reduced and Heuristics Testing" file contains the code used to run the experiments for the heuristics and their effect on the reduced model.
